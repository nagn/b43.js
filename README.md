What's this?
============

b43.js is a CoffeeScript port of GG2, or something.

Usage
=====

Create the `out` directory if it doesn't exist.

1. `./compile`
2. `cd out`
3. `python -m SimpleHTTPServer`
4. http://localhost:8000/
